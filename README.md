# Nonton
Mobile application for ordering a movie ticket. Written in kotlin and using firebase database.

## Features
1. Splashscreen
2. Onboarding
3. SignIn, SignUp dan Sign Out
4. HomeActivity
5. Fragment
6. Edit Profil pada DataBase
7. Preferences
8. Top Up
9. Buy Ticket

## Teknologi
1. Firebase
2. Kotlin
